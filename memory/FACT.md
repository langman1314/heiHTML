# 无光域项目 — 待办与记忆

## 待回答的本质问题（任务书完工后追问）

这四个问题是阅读全部设定后提出的深层世界观矛盾，待任务书七阶段全部完成后向用户提出：

1. **穿越者的"真实性"**：穿越者的故乡记忆是真实的（人被捕获），还是神胎根据某种人格模板植入的？如果是后者，那怀乡之情就是神胎设计的情绪生产线的一部分，这根本性地改变整个世界的情感底色。

2. **三重守恒的源头**：物质守恒、边界守恒、情绪守恒——这三条是谁定的？是神胎自身的生理规律，还是有某种更底层的规则在约束神胎本身？如果神胎是"未出生的神"，谁在制定约束神的规则？

3. **庇护所的悖论**：庇护所越像"家"就越活化，越活化就越危险（偏袒主人、隐藏问题、把外人喂给自己）。这是否暗示"家"这个概念本身在无光域中就是一种异常？穿越者试图重建"家"的行为本身就是一种污染？土著不建庇护所，他们怎么解决归属问题？

4. **终局路线的真实代价**：成神=人格被神胎吸收溶解还是获得控制权？留域=永远在黑暗中制度化生存直到情绪枯竭？出走=回到原世界还是被排到更可怕的未知？
