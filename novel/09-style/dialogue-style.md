# 对话风格规范

---

## 1. 基本要求

- 对话必须符合角色身份
- 必须有潜台词——角色不说的话比说出口的更重要
- 不能把设定直接说完——信息通过行动、环境、暗示传递
- 不能所有人说话都像作者本人——每个角色有自己的用词习惯和沉默方式

---

## 2. 晦民对话特征

### 沉默与低表达

- 晦民习惯简短回应，长句在他们文化中可疑
- 常见特征：省略主语、用动作代替回答、不直接说"不"
- 在公共场合压低声音，不完整句子是常态

**示例：**

> "那边。"（不解释"那边"是哪里）
> 
> "知道了。"（不等于"我会去"）
> 
> "你听到了。"（回避重复已经说过的话）

### 仪式化用词

- 对话中嵌入边界确认语
- 特定场合使用特定短语（如潮前/潮后/入夜/梦前）

**示例：**

> "门关好了。"（既是事实陈述，也是边界确认）
> 
> "床是你的。"（既是归属确认，也是风险声明）
> 
> "我数过了。"（既是报告，也是仪式完成声明）

### 非语言沟通

- 晦民大量使用动作、眼神、沉默传递信息
- 缄面族在公共场合几乎不用口头语言
- 轻叩门框、触摸床沿、摆正碗具——这些都是"话"

---

## 3. 穿越者对话特征

### 初期：地球语言习惯

- 口语、俚语、网感表达仍在
- 容易说太多，容易解释自己
- 不自觉用现代概念类比（"像游戏一样""跟外卖似的"）

**示例：**

> "这他妈到底是什么鬼地方？"
> 
> "等等，你说'名字'——是像账号那种意思吗？"
> 
> "所以这不就跟养宠物差不多吗……啊对不起。"

### 中期：开始吸收晦民习惯

- 不再轻易补全别人的话
- 学会用沉默回答
- 仍偶尔漏出地球习语（可以作为情绪失控的信号）

### 后期：被环境重塑

- 对话更短，潜台词更多
- 开始理解并使用仪式化语言
- 但保留了地球人的"追问"倾向（与晦民的"不问"形成张力）

---

## 4. 族群口音与用词差异

| 族群 | 对话特征 |
|------|----------|
| **骨纺人** | 务实、直接、多用建筑/编织类比。"墙会说""缝要收" |
| **壳眠裔** | 缓慢、含混，多用梦比喻。"我记得你做过……""你还没醒" |
| **缄面族** | 极简，能用动作不用语言。公共场合几乎不开口 |
| **食名裔** | 注重称谓和头衔，对话中频繁确认对方的"名字位置" |
| **裂瞳古裔** | 跳跃、抽象，能看到说话者看不到的东西 |
| **井喉民** | 多用井/水/回声比喻，声音有特定音调 |
| **灯骨匠裔** | 精确、技术性，喜欢用光/灯/影子的概念解释事情 |
| **灰蜕人** | 粗糙、直接，对禁忌的敏感度低——他们没什么好失去的了 |

---

## 5. 对话禁忌

1. **不要在对话中解释世界观**——"你知道，在我们无光域，门是很重要的边界器官……"
2. **不要所有角色用同样的句式长度和词汇复杂度**
3. **不要让反派说反派台词**——"呵呵呵，你们这些愚蠢的凡人……"
4. **不要让角色把情绪直白说出来**——"我很害怕"不如"他的手在抖，但他没去碰门"
5. **不要过度使用"他/她/它说"**——可以通过动作、视角切换来标识说话者
6. **不要用对话补充读者没看到的剧情**——"如你所知，刚才我们经历了一场大黑潮……"

---

## 6. 潜台词层级

| 说出的话 | 实际的意思 |
|----------|-----------|
| "床是你的。" | 我确认你属于这里 / 别睡错床 / 你的位置还在 |
| "门锁了。" | 我不希望任何人进来 / 我们安全 / 我在担心 |
| "我不饿。" | 我不想欠你的 / 食物不够分 / 我不信任你 |
| "你先睡。" | 我不确定你睡着了会怎样 / 我需要独处 / 我还不想放松警惕 |
| "知道了。" | 我不赞同但不想吵 / 我会考虑 / 别再说下去了 |

## 7. 特殊场景对话处理

### 黑潮期间的对话

- 必须用更低的声音（如果能在黑潮中听到正常音量的对话，说明黑潮强度低）
- 句子碎片化，以确认安全为优先内容
- 使用守夜骨铃、静皮帘等物理介质辅助交流

### 跨族群对话

- 双方都存在"被翻译"的损耗——对方的真实意思可能在文化框架下被曲解
- 穿越者容易把晦民的沉默理解为敌意
- 晦民容易把穿越者的直白理解为冒犯

### 死者/梦中的对话

- 梦腐层和旧梦中的对话边界模糊——说话者可能不是你以为的人
- 不要替死人把话说完
- 梦中的人物可能比现实中的人更了解你——但也可能只是"记得"了解你
