# 主线剧情

---

## 核心主线

穿越者（主角）坠入无光域，从最初的生存挣扎开始，逐步发现世界的真相——这是一个未出生的神胎内部，而所有穿越者都是投入其中的"新鲜人格材料"。主角的裂视能力让他看到系统过滤层之下的真实，也让他成为神胎最渴望的候选人格。

**主线的本质不是"升级变强"，而是"认知逐步渗透"和"选择逐步逼近"。**

---

## 主线三阶段

### 第一阶段：生存与认知（前1-50章）

**核心问题：这个世界到底是什么？**

- 主角首次经历黑潮，意识到这不是游戏
- 建立基础庇护所，接触第一个土著聚落
- 裂视能力觉醒，开始看到系统翻译之下的真实
- 初步了解五层胎腔结构、晦民文明、系统本质
- 遭遇第一个真正的选择困境（不是战斗，而是伦理）
- 埋设终局路线伏笔（接触某一神胎碎片的早期表现）

### 第二阶段：建设与冲突（51-150章）

**核心问题：在这个世界，人应该如何活着？**

- 聚落发展，从个体生存到集体建设
- 势力冲突展开（系统教、食名教团、饥饿教等）
- 裂视加深，开始看到神胎碎片的活动痕迹
- 接触上一轮外来者的遗产和教训
- 方舟概念出现，但路线争议开始
- 候选人格身份逐渐浮出水面
- 登舟筛选的伦理困境首次出现

### 第三阶段：终局选择（151章-终局）

**核心问题：当世界即将获得自我时，文明应该如何选择？**

- 五条终局路线全面展开，每条都有支持者和反对者
- 主角的方舟路线推进（妊娠四阶段）
- 崩坏计时器推进——犹豫的代价具象化
- 候选人格对峙（理念和制度的冲突）
- 方舟内部投票或公约（谁走谁留的伦理困境）
- 庇护所人格最终反应（方舟是否愿意成为方舟）
- 神胎第一次完整发声
- 穿越胎膜

---

## 主线与副线的关系

| 副线 | 与主线的关系 |
|------|-------------|
| 聚落建设 | 主线第二阶段的物质基础，方舟的起点 |
| 势力冲突 | 主线第二阶段的戏剧驱动力，终局选择的社会维度 |
| 裂视成长 | 认知渗透的机制，也是候选人格逼近的代价 |
| 角色关系 | 终局选择的情感维度——谁上船、谁留下、谁反对 |
| 遗产发掘 | 上一轮的教训，帮助理解当前轮的选择是什么 |

---

## 核心悬念

1. **世界的本质**——神胎、系统、晦民真相的逐步揭示
2. **主角的身份**——为什么他适配多个碎片？他的穿越是随机还是被选中？
3. **终局的走向**——五条路线谁会胜出？主角能否避免崩坏？
4. **方舟的结局**——外渊到底是什么？方舟会不会变成新神胎？
5. **晦民的未来**——被消化过又被吐出来的文明，是否有真正的出路？
