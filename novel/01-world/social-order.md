## 聚落承载力四原则

1. **就地取材原则**——聚落规模由周边可稳定获取的资源决定，不能依赖远距离补给维持基本生存
2. **低熵优先原则**——更简单的生存方式（骨麦种植 + 凝水收集）比复杂方式（跨层贸易 + 晦能精炼）更可持续
3. **功能残留原则**——聚落废弃后，其边界关系不会立即消失——空屋有人等、旧门有人敲
4. **转化有损原则**——任何资源转化都有损耗（骨麦→食物损耗20%，晦能→照明损耗40%）

## 聚落十级体系

| 等级 | 人口 | 特征 |
|------|------|------|
| **一级** | 1-5人 | 孤门庇护所——单一家族或少数幸存者 |
| **二级** | 5-30人 | 小型庇护所群——多家庭协作，初步分工 |
| **三级** | 30-300人 | 灰醒层小聚落——开始有简单制度和行业 |
| **四级** | 300-800人 | 灰醒层集镇——专业工匠出现，定期集市 |
| **五级** | 500-5000人 | 骨沉层城镇——完整五庭制度，行业分工完善 |
| **六级** | 5000-20000人 | 骨沉层城邦——多法庭并存，跨层贸易网络 |
| **七级** | 500-10000人 | 混合玩家领地——玩家与土著混居，系统法+边界法并存 |
| **八级** | 3000-30000人 | 活化领地群——庇护所开始活化，领地有自我意识 |
| **九级** | 5000-50000人 | 方舟胚胎共同体——以方舟建设为共同目标的文明群 |
| **十级** | 10000-100000+人 | 子宫方舟——最大的社会组织形式，活体方舟 |

## 社会阶层

### 土著阶层

| 阶层 | 组成 | 社会地位 |
|------|------|----------|
| **名权阶层** | 拥有完整真名、名册位置稳定的家族 | 上层——参与政治决策 |
| **匠裔阶层** | 掌握核心技术的工匠家族（骨纺、灯骨、井喉） | 中上层——不可或缺 |
| **自由晦民** | 有名字有床位但无名权背景的普通人 | 中层——聚落主体 |
| **脱册者** | 高风险职业从业者，出发前从名册除名 | 边缘——活着但不被计数 |
| **半名者** | 半名期儿童、名削刑罚中的人 | 底层——法律地位不完全 |
| **无名人** | 永久无名者或被食名者 | 不存在——不被任何边界承认 |

### 玩家阶层

| 阶层 | 特征 |
|------|------|
| **系统贵族** | 高排名玩家，利用系统身份体系巩固权力 |
| **领地领主** | 建立或控制了聚落的玩家势力 |
| **自由玩家** | 未加入大势力，独自或小团队探索 |
| **新手穿越者** | 刚刚投放，未经历第一次黑潮 |

## 教育体系

### 儿童教育

- **半名期（0-7岁）**——仅使用乳名，不建立完整名册索引
  - 教育内容：基本生存技能、边界识别、危险词识别
  - 核心规则：不可在儿童面前讨论食名、梦交易、死亡细节

- **全名期（7岁后）**——获得真名，进入名册
  - 教育内容：专业技艺传承、语言深入学习、法律基础
  - 成人礼：第一次独立经历黑潮

### 职业教育

- 师徒制为核心
- 族群专长技能由族群内部传承
- 跨族群技艺通过商队和混合聚落传播

## 家庭结构

- **核心单位**：床（不是房子或家庭）——床是"家"的最小边界单位
- **扩展关系**：通过门（通行关系）和名（索引关系）连接多个床
- **家庭仪式**：晨起点名、入床秩序、开门确认、归位
- **家庭禁忌**：不可在床前争执、不可在门附近食言、不可在井边撒谎

## 脱册制

脱册制是无光域独特的社会机制。

### 定义

高风险职业从业者（井喉深潜者、骨沉层清道夫、黑潮前探人）在出发前从聚落名册除名。

### 代价

- 存在但不被计数
- 活着但不被承认
- 名字听不见回应
- 床觉得他们陌生
- 门犹豫是否让他们通过

### 长期后果

脱册者会慢慢变成"半个晦诡"。灰蜕者就是积累了多年的老脱册者。

### 穿越者的特殊性

穿越者天生不在无光域名册上 = 天然脱册者。因此他们可以做更多冒险，但也被同化得更慢。这解释了为什么穿越者能在初期做到土著做不到的事——也解释了为什么他们不理解土著为什么"不敢"做。
