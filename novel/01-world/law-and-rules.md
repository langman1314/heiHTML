# 法律与规则

来源：`phase4-law.js`

---

## 法律第一性原理

无光域的法律不是源于立法者，而是源于**边界关系**。

法律的本质不是"谁有权制定规则"，而是"哪些边界关系需要被稳定承认"。因此：

- 法律不是被"制定"的——是被发现、被记录、被实践的
- 法律不是关于惩罚——是关于维持边界不崩
- 法律不是人独有的——门、床、名、井都是法律主体

---

## 三证原则

法律证据必须同时满足三项才能成立：

1. **系统记录**——系统的行为日志（可被伪造，可被错栏干扰）
2. **物理痕迹**——骨刻、床纹、门痕、井脉变化（需要专家解读）
3. **边界承认**——门、床、名、井对事件的"表态"（最高证据）

**三证俱全 = 铁案。缺一项 = 可上诉。缺两项 = 存疑。缺三项 = 不成立。**

## 六庭体系

| 法庭 | 管辖范围 | 核心原则 |
|------|----------|----------|
| **名庭** | 名字、身份、名册、食名纠纷 | 一个名字对应一个现实座位 |
| **门庭** | 通行权、门路归属、边界争议 | 门内外是不同的现实，门记得每一次通行 |
| **床庭** | 居住权、家庭关系、庇护所归属 | 床承认谁，谁就属于这个家 |
| **井庭** | 水源权、公共资源分配 | 井脉属于所有人，但使用权需要边界承认 |
| **骨庭** | 遗产、葬制、死者权利、骨刻证据鉴定 | 死者有边界残留，活人无权单方面替死者决定 |
| **讯庭** | 紧急状态法庭（黑潮期间或灾后即时裁决） | 效率优先，但裁决可在潮后上诉 |

### 讯庭的特殊性

讯庭是唯一在流程上简化的法庭形式。发生在黑潮期间或灾后，必须快速裁决。讯庭裁决可在潮后30天内向正式法庭上诉。

## 审判七步流程

1. **启动**——争议一方提交申请，法庭确认管辖权
2. **陈述**——双方陈述事实，各自援引法律依据
3. **三证举证**——系统记录、物理痕迹、边界承认逐一呈现
4. **裂视读证**——裂视者对争议边界进行官方解读（裂视者的解读具有法律效力）
5. **辩论**——法官与各方就法律适用进行辩论
6. **裁决**——法庭做出裁决，写明法律依据和三证匹配情况
7. **执行与备案**——裁决执行，录入名册和系统日志

## 七种刑罚

| 刑罚 | 内容 | 严重程度 | 可逆性 |
|------|------|----------|--------|
| **名削** | 从名册中暂时移除 | 轻 | 可逆（缴纳罚名或完成劳役后可恢复） |
| **床冷** | 剥夺床位承认 | 中 | 可逆（视罪行轻重和时间） |
| **门禁** | 限制通行权（时限制或区域制） | 中 | 有限可逆（门会记住违规者） |
| **逐出** | 从聚落驱逐 | 重 | 有限可逆（需全聚落重新承认） |
| **葬制拒绝** | 死后不得葬入聚落墓地 | 极重 | 不可逆（死后无葬 = 死后无边界） |
| **空名** | 名册中保留位置但内容清空 | 极重 | 几乎不可逆 |
| **永久无名** | 从所有名册中彻底删除 | **文明禁忌** | **不可逆**（极少使用，仅用于最严重罪行） |

### 永久无名的特殊性

永久无名是整个法律体系中最具争议的刑罚。将一个存在从所有名册中彻底删除，意味着这个人：

- 不再被任何门承认
- 不再被任何床接受
- 名字无回应
- 死后无葬、无梦存、无骨存

它如此残酷以至于它本身就是"文明禁忌"——法律禁止自己过度使用最重的刀。

## 死亡确认制度

在无光域，死亡不是瞬间事件，而是一个**需被确认的流程**。

### 八种死亡形态

| 形态 | 说明 |
|------|------|
| 清死 | 尸体完整，名册可正常注销 |
| 缺死 | 尸体不完整（部分被消化），需骨庭鉴定 |
| 错死 | 系统记录死亡但人还在，或人已死但系统未记录 |
| 活死 | 身体存活但名册/床/门已不承认其为生者 |
| 梦死 | 在梦中死亡，现实身体无损伤但无法唤醒 |
| 名死 | 名册上被删除但身体尚存 |
| 床死 | 床不再承认其为睡眠者，即使人还活着 |
| 未断死 | 死者已确认但仍有未关闭的关系（旧主未断） |

### 死亡确认三步骤

1. **身体确认**——尸体查验或不可唤醒确认
2. **名册确认**——名册注销，同时检查名册是否"拒绝注销"（名存现象）
3. **边界确认**——床、门、物品的归属关系逐一切断

## 庇护所财产法

庇护所内的财产归属遵循"**使用 + 边界承认 = 所有**"原则：

- 单纯持有不等于所有——还需要床和门的承认
- 无主物 = 无边界承认之物，需通过法庭重新建立归属
- 庇护所的墙、门、床本身是法律主体，不是财产

## 六条基本法

1. **边界不可侵犯**——未经承认的边界跨越是最高罪行
2. **名不可替**——未经本人同意的身份替代是文明禁忌
3. **死者有残留权利**——活人无权单方面替死者做决定
4. **床不欺主**——庇护所不得主动欺骗居住者
5. **门不隐路**——门不得隐藏自己的通行记录
6. **井不独属**——井脉是公共资源，不可私有化
