## 正史第一条：神胎意识状态

### 作者正史答案

**神胎没有成熟自我意识，但正在形成意识。**

神胎本体是一个巨型代谢/孕育系统。它不是一个已经醒来的神，而是一个正在通过世界、晦民、候选人格和六神胎碎片"孕育人格"的胚胎性存在。

- 神胎本体 = 巨型代谢/孕育系统（已完成）
- 神胎意识 = 尚未完成的胚胎意识（正在进行）
- 六神胎碎片 = 意识形成前的六种本能/器官逻辑（分化中）
- 候选人格 = 神胎寻找自我意识的候选模板（系统启动后开始）
- 晦民 = 被神胎代谢系统改写后的细胞化生命（已完成）
- 黑潮 = 神胎周期性消化/筛选/供能现象（持续运行）

### 对全文结构的含义

1. 故事不是"人类反抗神明"——因为神尚未诞生
2. 故事不是"人类对抗灾害"——因为灾害背后有孕育中的主体
3. 故事是"人类在决定一个神会不会诞生、以什么人格诞生、由谁成为它的意识核心"

### 破胎行为的真实性质

- 不是单纯杀神（神还不算活着）
- 不是单纯逃灾（系统有逻辑而非纯随机）
- 而是**中止一个神的出生，或者改写它的出生方式**

### 终局核心伦理问题

如果神胎诞生，现有人类文明会被吸收。
如果破胎成功，世界可能失去维持结构的母体。
如果让某个候选人格成为神胎意识，等于把整个世界交给一个人。

## 正史第二条：系统来源

### 作者正史答案

**系统不是神胎原生器官，而是上一轮文明在门脉层中留下的"翻译层 + 育种协议 + 逃逸筛选器"。**

系统最初是为了理解神胎、筛选破胎者、寻找方舟适格者而建立。但在长期运行中被神胎代谢逻辑污染，变成了半工具、半寄生协议。

- 系统来源：上一轮文明
- 系统载体：门脉层
- 系统初始功能：翻译神胎规则、筛选候选人格、诱导适格者成长
- 系统被污染方式：被神胎的育种逻辑反向侵蚀
- 系统现状：不是完全忠于人类，也不是完全忠于神胎

### 系统的本质定位

系统**没有人类意义上的人格**。系统有**目标函数**。

- 目标函数最初来自上一轮文明
- 目标函数后来被神胎育种逻辑污染
- 因此系统既会保护玩家，也会把玩家推向更残酷的筛选

### 对全文结构的含义

- 前期：读者以为系统是外挂
- 中期：发现系统是筛选器
- 后期：发现系统本身也是上一轮文明失败后的残骸
- 终局：发现系统一直在寻找"既能破胎，又能不变成新神胎"的人格

## 正史第三条：六神胎碎片关系

### 作者正史答案

**六神胎碎片不是六个独立神，也不是六个完全分裂的个体，而是同一个未成熟神胎在不同层级、不同组织、不同人类认知中显现出的六种本能逻辑。**

它们是**同源异相**。

| 碎片 | 本质 | 解释 |
|------|------|------|
| 饥饿碎片 | 摄取本能 | 神胎的进食冲动 |
| 母巢碎片 | 繁殖本能 | 神胎的保护/圈禁冲动 |
| 名位碎片 | 秩序本能 | 神胎的命名/分类冲动 |
| 梦腐碎片 | 记忆本能 | 神胎的记忆/梦境/腐化冲动 |
| 门脉碎片 | 神经本能 | 神胎的传导/连接冲动 |
| 方舟碎片 | 排异本能 | 神胎的逃逸/保种冲动 |

它们不是六个神胎互相打架，而是神胎尚未形成统一意识前的**六套底层冲动**。

### 候选人格的真实含义

- 候选人格不是选择六个碎片之一
- 候选人格是在六种本能之间建立主导结构
- 谁能整合六种碎片，谁就有资格成为神胎意识的核心

### 终局路线的重新定义

- 失败路线：某一碎片独占主导，神胎畸形诞生
- 方舟路线：拒绝整合，带部分生命逃离胎体
- 破胎路线：切断六碎片循环，阻止神胎出生
- 主角路线（留域+方舟复合）：理解六碎片关系，选择既不成为神、也不让世界继续被孕育系统吞噬的第三道路

### 人类误解的来源

六神胎碎片是同一个未完成意识的六个功能性偏向。它们互相冲突，但冲突不是战争，而是人格形成前的内部争夺。人类误以为它们是六个神——那只是人类对神胎不同器官逻辑的宗教化解释。

## 正史第四条：上一轮文明残留

### 作者正史答案

**上一轮文明没有完全消失。它的肉身文明被神胎消化，但有三类残留未被完全消化。**

### 三类残留

1. **物质残留**：旧门、遗迹、方舟残骸、无法被晦能完全分解的材料
2. **信息残留**：系统协议、门脉记录、异常梦境、失真历史
3. **意识残留**：上一轮文明中少数人的人格碎片，以门脉回声或守墓者形式存在

### 旧门守墓者的真实身份

旧门守墓者守护的不是单纯的门，而是**上一轮文明最后一次失败的现场**，以及门后尚未被神胎完全消化的历史残骸。

### 对全文结构的含义

世界不只是"当前人类遇到灾难"，而是**很多代文明都在同一个胎体里挣扎过**。
每一轮文明都以为自己是第一批发现真相的人。
实际上他们只是神胎漫长孕育周期中的一次次代谢波峰。

### 上一轮文明的结局

上一轮文明曾经接近破胎，但失败了。
他们没有完全灭亡，而是被分解成门脉层中的记忆残响。
系统是他们失败前留下的协议。
旧门守墓者是部分残留意识与后代组织共同形成的守护势力。
他们守护的不是出口，而是"失败不能被遗忘"的证据。

## 正史第五条：外渊定义

### 作者正史答案

**外渊不是天堂，也不是虚无，而是胎膜之外的未知荒域。它不是"无晦能之地"，而是"低晦能/异晦能/非胎内规则区"。**

外渊确实存在，但方舟派对它的理解是不完整的。

### 方舟派的证据（残缺但真实）

1. 上一轮文明留下的外渊信标
2. 门脉层中记录过"胎膜外低晦能噪声"
3. 某些旧门短暂打开时，曾带回非胎内物质
4. 方舟派掌握过一段残缺回声，证明有人曾短暂抵达胎膜外
5. 但没有任何完整归来者

### 外渊的真实状态

- 外渊存在
- 外渊不受神胎完整控制
- 外渊不是安全之地
- 外渊没有稳定生态
- 外渊可能有其他胎体、尸胎、空壳世界、旧文明残骸

### 方舟派的真实逻辑（不是盲信）

留在胎内 = 迟早被神胎吸收
破胎 = 可能导致世界结构崩塌
夺取神胎人格 = 可能诞生新暴君
方舟出逃 = 风险极高，但至少不是在神胎规则内下注

方舟派不是蠢，也不是邪教，而是一种**极端理性的逃亡主义**。

## 五条正史的关系拓扑

```
神胎（无意识，正在形成意识）
 ├── 代谢系统 → 黑潮、晦能循环、五层胎腔
 ├── 六种本能 → 六神胎碎片（同源异相）
 ├── 候选模板 → 系统筛选（来自上一轮文明）
 └── 边界残余 → 上一轮文明残留（物质/信息/意识）

外渊 = 胎膜外的未知荒域（不是答案，是新的问题）
```

这五条一旦确定，全文主线形成明确递进：

1. 第一阶段：主角以为黑潮只是灾难，系统只是外挂
2. 第二阶段：主角发现世界像一个活体结构，神胎不是传说
3. 第三阶段：主角发现六神胎碎片不是六个神，而是同一存在的不同本能
4. 第四阶段：主角发现系统来自上一轮文明，但已经被污染
5. 第五阶段：主角发现上一轮文明曾经失败，方舟路线有真实依据
6. 第六阶段：主角发现神胎意识尚未完成，候选人格竞争的真相浮出水面
7. 终局阶段：主角必须决定是破胎、逃离、成为神胎人格，还是创造第四条路
