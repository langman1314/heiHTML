## 主角第一性定位

他不是黑暗中的纯洁火种，而是一个看见黑暗结构之美后，仍试图不让自己的审美变成他人灾难的人。

### 核心驱动力三层次

| 层次 | 内容 |
|------|------|
| 第一层·求生欲 | 初期和其他玩家一样需要水、食物、门、床、晦晶、庇护所。但不同于普通玩家——他第一次看见一种足以让他想继续观看的巨大未知。 |
| 第二层·审美欲 | 穿越瞬间看见"造主"后，审美被永久改变。会被晦诡生成链的精巧、庇护所从房子变成家的过程、黑潮像世界呼吸一样吞吐资源所震撼。这也是危险源。 |
| 第三层·文明欲 | 中后期不再满足于个人求生。意识到单人活下去没有意义——让文明获得不被神胎唯一解释的权利。方舟路线真正的动机。 |

### 主角不是传统救世主

- 没有干净的资源、没有无害的安全、没有不带代价的拯救、没有完全不伤人的文明选择。
- 真正要面对的不是"如何不做坏事"，而是：当每一种活法都要付出代价时，哪些代价绝不能让别人替自己在不知情中支付？

### 具备反派资格

- 与主要反派的区别不在于能力，而在于是否承认他人的边界。
- 能理解反派为什么会那样想——所以堕落不是突然黑化，而是逐渐觉得"也许他们只是比我更诚实"。

## 主角三条底线

### 底线一：不允许自己被系统替代思考

可以使用系统，但不能让系统决定何为正确。必须始终区分：系统奖励的行为、生存有效的行为、文明可接受的行为、自己愿意负责的行为。

若越界→变成系统教式人物：高效、理性、强大，却把人变成系统路线上的材料。

### 底线二：不允许庇护所吞噬无辜者成为常规燃料

可以使用腐殖、尸体、晦兽、污染物、黑潮残余来养房。但不能让庇护所把活人、弱者、俘虏、欠债者、外来者常规化当作成长资源。

若越界→变成饥饿教或巢穴型方舟主，仍可能拯救很多人，但他的文明从第一天起承认"有些人可以被系统性地吃掉"。

### 底线三：不允许神胎、系统或方舟剥夺个体选择权

必须保护个体在是否登舟、是否留域、是否接受晦染、是否卖梦、是否借名等问题上的最低选择权——知情、可撤回、可被见证、不被债务或宗教或系统胁迫。

若越界→变成母巢型救世主：温柔、保护、强大，但不再相信别人有资格决定自己的命运。

## 主角可接受行为清单

| 行为 | 前提 |
|------|------|
| 利用怪物与尸体循环 | 死者关系处理完整、不主动制造无辜死亡、不隐瞒资源来源、不让活人燃料化成为制度 |
| 利用晦诡 | 不故意让公共晦诡成熟、不用无辜者测试触发条件、不把高危晦诡卖给他人、不为观察完整形态而延迟救援 |
| 欺骗敌人 | 应针对明确敌对者，而非公共现实。不伪造他人真名、不利用无辜者身份作诱饵 |
| 保密 | 隐藏裂视信息、方舟结构、神胎真相。保密的理由必须是防止灾害，而非操控他人选择 |
| 做残酷选择 | 承认被放弃者不是"没价值"、不把残酷选择包装成美德、事后承担责任、不让特殊情况变成常规制度 |

---

## 主角绝不接受行为清单（六道硬边界）

1. **不接受食名式替代**——即使替代者更有能力，也不能抹掉原主身份。
2. **不接受儿童初梦交易**——儿童初梦、半名期真名、第一床位不可市场化。
3. **不接受非知情献祭**——自愿牺牲必须有完整知情、可撤回窗口、见证、不被债务胁迫。
4. **不接受永久性无名惩罚**——不能接受把一个人永久从现实座位上抹掉。
5. **不接受方舟资格完全资产化**——不能让登舟权只由财富、排名、技能决定。
6. **不接受庇护所替居民做最终人生选择**——庇护所可以预警，但不能决定谁永远不能离开。

## 堕落路线（六条堕落镜像）

1. 系统化领主——高效管理，把人变成数据
2. 饥饿循环——相信一切终将被消化
3. 方舟筛选——为"多数人"牺牲"不适合者"
4. 母巢保护——温柔剥夺选择权
5. 审美沉溺——为观赏结构之美而延迟阻止灾难
6. 成神——成为神胎主导人格，失去"人"的可申诉性
