# 能力使用代价

---

## 裂视代价体系

| 等级 | 使用代价 |
|------|----------|
| 0级 | 头痛、耳鸣、睡眠质量下降 |
| 1级 | 精神疲劳、梦中重复标签 |
| 2级 | 自身名字偶尔变轻、庇护所依恋增强 |
| 3级 | 梦境污染、被同类晦诡反向识别 |
| 4级 | 路会记住主角、归路发芽概率上升 |
| 5级 | 每次裂视改变自身候选倾向 |
| 6级 | 不可完全恢复，一部分自我被神胎记住 |

## 系统的代价

1. 使用系统越深入，越被系统定义——行为模式被诱导
2. 排行榜上榜越高，越容易被位置吞掉
3. 依赖系统翻译，会忽略土著知识——死得快
4. 新手保护会随能力成长失效
5. 系统任务可能诱导高回流行为——违背道德

## 庇护所的代价

1. 庇护所越活化，越难离开——依恋增强
2. 庇护所可能开始"劝留"主人
3. 庇护所会学习主人的恐惧——反向利用
4. 庇护所吞并废屋可能带回旧主污染
5. 过度庇护形成"温柔牢笼"——个人边界消失

## 情绪代价

1. 恐惧被制度化后产出下降——不再恐惧的人不再优质
2. 情绪被反复压榨后变得麻木——枯竭回流
3. 强烈、无序、未封存的情绪会被晦能采样
4. 失去名字的代价：现实索引失效
