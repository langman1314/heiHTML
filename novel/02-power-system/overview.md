## 系统第一性定义

系统是神胎生成的外来者接入协议，完成六大功能：翻译、保护、诱导、采样、筛选、人格整合。

系统不是谎言，但永远不是完整真相。三种不完整：
1. 信息不完整——只显示系统能编码的内容
2. 定义不完整——把复杂边界压缩为简单标签
3. 代价不完整——不显示行为的完整边界代价

## D = C × A 公式

### 诡异度公式

**D（诡异度）= C（胎腔层浓度因子）× A（载体亲合系数）**

- C值由植物所在的胎腔层深度决定——灰醒层最低，胎海层最高
- A值的判定：移走所有观察者，它照长不误→A≈0（务实植物）；不照长→A>0（诡植物）

### 三个公理

1. **谱系律**——所有晦植必须在务实谱系（A≈0）与诡谱系（A>0）之间落位。不存在"既是务实又是诡"的中间态。
2. **正交律**——潮级（避潮→门深）与触发级（常触→危触）是两个独立的分类轴。一株植物同时拥有一个潮级和一个触发级，互不冲突。
3. **回流律**——食物网必须闭合。胎海是回流口。

## 裂视能力

### 裂视的第一性定义

裂视不是系统赐予的能力，而是系统过滤层破裂的伤口。穿越瞬间看见"造主"后，普通系统晦视接口被冲裂产生的异常感知。

### 裂视能看见的三类事物

1. **未完成定义**——正在成形中的晦诡、尚未完全活化的庇护所、正在认主的物品
2. **错误标注**——系统、名册、门牌、排行榜等记录与现实边界不一致
3. **边界缝隙**——门缝中不是空间而是通行关系、床与睡者之间的归属线变细

### 裂视的代价

| 代价 | 内容 |
|------|------|
| 被看见 | 看得越深，越容易被深层看见 |
| 边界削薄 | 自我与世界之间的皮肤变薄 |
| 梦境污染 | 梦不再是私人空间 |
| 系统标注异常 | 新手保护失效 |
| 庇护所依恋增强 | 庇护所开始不愿主角远行 |
| 候选人格靠近 | 神胎碎片投射诱惑 |

### 裂视等级

| 等级 | 能力 | 代价 |
|------|------|------|
| 0级·裂痕残光 | 偶尔看见系统标签闪烁 | 头痛、耳鸣 |
| 1级·错标识别 | 发现系统标注与现实不一致 | 梦中重复标签 |
| 2级·边界线感知 | 看见归属线 | 名字变轻 |
| 3级·规则注释 | 读取残缺骨刻语 | 梦境污染显著 |
| 4级·缝隙通行 | 发现隐藏门路 | 路会记住主角 |
| 5级·候选注视 | 识别其他候选人格纹路 | 神胎碎片开始回应 |
| 6级·临胎裂视 | 看见无光域整体代谢 | 不可完全恢复 |

## 庇护所活化机制

庇护所不是房屋，而是系统、文明与神胎争夺"家"概念的中间物。

### 三种结构叠合

1. **系统安全接口**——初始生存保护点
2. **文明边界钉**——门、床、箱、灯、灶将旧世界秩序钉入无光域
3. **晦能活化器官**——吸收晦晶、尸体、恐惧，逐渐长出类似生命体的反应

### 三种权

- **占有权**——系统层面的操作权
- **居住权**——门床箱墙的承认
- **内属权**——房屋将人视为自身一部分的危险状态

### 活化阶段（0级→7级）

从死屋到子宫方舟，每阶段都在深化人与房屋的边界重叠。
