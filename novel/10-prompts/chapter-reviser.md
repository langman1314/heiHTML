# 章节润色提示词

请对刚完成的章节进行润色优化。

## 润色目标

1. 强化画面感。
2. 消除 AI 腔。
3. 优化对话自然度。
4. 调整节奏。
5. 强化情绪张力。
6. 删除多余的说明性文字。

## 检查清单

- [ ] 段落长度是否过于平均。
- [ ] 是否频繁使用"仿佛、似乎、好像"。
- [ ] 是否有不必要的心理描写。
- [ ] 对话是否符合人物身份。
- [ ] 环境描写是否对情节有用。
- [ ] 章节结尾是否有钩子。
- [ ] 是否有大段设定解释。
- [ ] 是否有"他不知道的是"这类上帝视角。
- [ ] 是否有"命运的齿轮"这类陈词滥调。
- [ ] 是否每段开头句式雷同。

## 输出格式

```
# 章节润色报告

## 修改清单

| 位置 | 问题 | 修改后 |
|------|------|--------|
| 第X段 | 问题描述 | 修改文本 |

## 整体建议

-
```
