# 专有名词与语言体系

来源：`phase3-language.js`

---

## 系统翻译第一性原理

系统翻译不是语言学翻译，而是**生存行动翻译**。

系统翻译优先让玩家"能行动"而不是让玩家"真正理解"。它会把复杂的土著词压缩成玩家熟悉的简单词，导致玩家经常以为自己听懂了，但实际只理解了表面。

> 玩家最大的危险不是听不懂土著，而是他以为自己听懂了。

---

## 六大翻译缺陷

| 缺陷 | 说明 | 示例 |
|------|------|------|
| 压缩翻译 | 多个土著词压缩成一个玩家词 | 十种"门"全部翻译为"门" |
| 安全化翻译 | 用温和词替换高危词 | "床在等上一位睡者"→"存在旧主残留" |
| 游戏化翻译 | 将危险描述转化为增益描述 | "屋子把你咽到里面半寸"→"庇护所活化度提升" |
| 去历史化翻译 | 省略词背后的灾难史 | "潮前换床"被翻译但不说原因→迷信化 |
| 去边界化翻译 | 将边界词翻译成普通物品词 | "碗"=共食资格容器→翻译为"碗" |
| 延迟纠错翻译 | 随权限提高逐步修正翻译 | "门"→"旧门"→"旧主未断、曾吞通行关系之门" |

## 十大高危误译词

| 词 | 系统初译 | 土著实际分类 | 误解后果 |
|----|---------|-------------|---------|
| **门** | 门 | 生门、旧门、饱门、认主门、回人门、内开门、假门、门尸、未门 | 门灾、错路、敲门者入侵、叠室化 |
| **家** | 家、住所、庇护所 | 暂居处、栖屋、认门之家、认床之家、内属之家、等人之家、饥饿之家、巢之家、可迁之家 | 被过度内属、忽略巢穴化 |
| **死** | 死亡 | 清死、缺死、错死、活死、梦死、名死、床死、未断死 | 频道遗言、晚归影、活死者、遗物污染 |
| **名字** | 名字、昵称、ID | 乳名、真名、门名、床名、榜名、借名、旧名、死名、轻名、空名 | 食名、失名、榜首失名 |
| **梦** | 梦、精神状态 | 可食梦、可存梦、可借梦、可葬梦、禁述梦、不可醒梦、反梦、旧梦、归梦、空梦 | 梦污染、替睡者、不可醒梦 |
| **交易** | 交易、买卖、交换 | 物换、断主换、带梦换、带名换、夜声换、半断换、债换、见证换、禁换 | 旧主回归、梦债、物品自返 |
| **安全** | 安全、危险等级低 | 暂安、门安、床安、名安、梦安、物安、灯安、假安、过安 | 正午错觉、伪安全灯、门区松懈 |
| **净化** | 净化、清除污染 | 水净、火净、骨净、静净、梦净、名净、屋净、断净、缓净、伪净 | 名污染残留、净化物再出声 |
| **活化** | 活化、升级、增强 | 醒材、认主活、习惯活、饥饿活、巢活、旧活、多心活、方舟活 | 饥饿屋、巢穴化、假方舟 |
| **灯种** | 外来者、玩家、穿越者 | 携带光记忆者、可被点燃的燃料、可能照亮新路的人、可能重复旧灯灾的人 | 低估土著对玩家真实处境的了解 |

## 十个不可直译概念

| 概念 | 表层翻译 | 实际含义 |
|------|---------|---------|
| 归位 | 回到位置、整理物品 | 让人物床名碗箱门重新被现实承认为各在其位——不是整理，是防止世界重新解释你属于哪里 |
| 不等 | 不等待 | 对空床空屋进行封闭声明——不是冷漠，是防止空缺继续长出人形 |
| 暂呼 | 临时称呼 | 半名期儿童只能被有限呼唤，不可建立完整现实索引 |
| 旧主未断 | 旧主人残留 | 物品、床、门、名字仍保留回到前归属关系的路径——不是残留物，而是未关闭的关系 |
| 床冷 | 床铺温度低 | 床不再承认某人为安全睡眠者——可能比身体受伤更严重 |
| 门饱 | 门异常 | 门吞过太多通行关系，不应再承担门的功能——再开不是开门而是喂路 |
| 梦醒了 | 做梦者醒来 | 梦中某个存在意识到自己是梦，开始寻找进入现实的路径 |
| 名轻 | 名字不稳定 | 现实通过名字索引个体的力量下降，别人更容易替代、误认、借名 |
| 屋子撒谎 | 庇护所异常 | 庇护所面板、门、床、墙对同一事实给出不一致反应，且房屋有意隐藏某些边界变化 |
| 潮后未结 | 黑潮后状态未结束 | 系统倒计时上结束，但区域代谢、死者关系、资源归属、梦污染仍未完全闭合 |

## 语言学习七级

| 等级 | 能力 | 风险 |
|------|------|------|
| 0级 | 完全依赖系统翻译——把门都当门、把死都当死 | 风险最高 |
| 1级 | 识别高危词——知道旧门不等于普通门，空床不等于可用床 | 仍然依赖系统 |
| 2级 | 理解边界语法——听出一句话是在确认边界还是在拒绝承认 | 可发现基础矛盾 |
| 3级 | 阅读骨刻与灯纹基础——可独立低风险探索，能发现系统翻译遗漏 | 进入深层理解 |
| 4级 | 理解仪式语言——可参与混合法庭与庇护所管理深层剧情 | 接触危险知识 |
| 5级 | 识别系统误译——判断系统把哪些词压缩了、安全化了 | 成为玩家-土著桥梁 |
| 6级 | 危险直听——不经系统过滤直接理解深层语言 | 未过滤的语言可能直接污染梦与名 |

## 语言六律

1. **系统让你听懂，不等于让你懂**——翻译的是行动所需表层词，不是文明分类
2. **一个玩家词可能压着十个土著灾难词**——门、床、死、名、梦、家都不是单一概念
3. **误译不是沟通失败，而是边界事故**——说错听错译错可能让门被误开、死者被误召
4. **语言越精确越能活，语言越完整越危险**——精确分类能防灾，完整复述会喂养异常
5. **谁掌握词，谁掌握现实解释权**——系统、土著专业者、玩家攻略者都在争夺命名权
6. **新文明必须发明能保护人的语言**——方舟必须带走一套知道何时命名、何时沉默、何时遗忘的语言

> 在无光域，真正学会一门语言，不是知道"门"怎么说，而是知道哪一种门不能开；不是知道"死"怎么说，而是知道一个人到底要断掉多少关系，才算真正离开。
