19区域地图与探索分层

前期地图围绕庇护所形成近、中、远三圈；中后期则依世界五层逐级深入。每一层都同时提供资源、危险、势力接触与真相推进。

灰醒层区域

孤门空地、灰木林、湿壁裂谷、白骨洼地、眠菇洞、灰钟坡、褪灯池、门苔坡。

骨沉层区域

折梁谷、百骸埠、活墙庄园遗迹、旧灯堆肥场、骨桥洄口、沉箱地窖、驮壳滩。

梦腐层区域

回忆沼、孵梦林、浅眠洼地、眠渊、旧梦藻池、回声井群、共梦坟场。

门脉层区域

锈门坡、叠室遗迹、空白公告厅、断链门廊、错栏庭院、初筛回廊、箱契长廊。

胎海层区域

胎潮岸、脐索峡、脐井口、未诞大厅、晦心群岛、初门近域。

代表区域图鉴

灰钟坡

所属层级：灰醒层

地貌：风吹骨腔会发出远钟般的闷响，地表遍布中空碎骨与低矮灰石。

资源：灰石、骨粉、少量门槛苔、裂穗骨麦野生种。

威胁：追声型晦兽活跃，尤其容易引来听骨犬与沼喉犬。

剧情功能：适合前期制造第一次“声音会招来东西”的认知转折。

折梁谷

所属层级：骨沉层

地貌：遍地旧屋梁架、骨柱与坍塌结构，像一片被压扁的废弃聚落。

资源：高级灰木、旧屋构件、活墙残片、门栓材。

威胁：吞架熊、缝背狸、空箱占用者残留事件。

剧情功能：适合主角第一次尝试回收废屋，理解“房子也会留下尸体”。

活墙庄园遗迹

所属层级：骨沉层

地貌：一座仍在微弱生长的旧庄园，墙体表面会缓慢收缩，像在睡眠中换气。

资源：活墙残片、活化钉原料、旧门家具、房屋记忆碎片。

威胁：未归床位、存档床单、门后换班者。

剧情功能：让“房子会不会活过来”第一次变得具体。

回忆沼

所属层级：梦腐层

地貌：泥泞来自别人的人生片段。每一步都可能踩进一段不属于自己的旧日生活。

资源：记忆碎片、旧梦藻、梦渣浓缩块、梦囊材料。

威胁：倒影迟到者、旧灯影、替睡者残留。

剧情功能：适合揭露上一轮外来者历史，也适合主角裂视升级。

叠室遗迹

所属层级：门脉层

地貌：房间套着房间，门后仍是门。有些房间属于不同年代、不同主人、不同规则系统。

资源：规则碎片、系统残页、门骨架、房间命名术线索。

威胁：多出的任务、错栏公告、多出来的门内侧。

剧情功能：揭示系统并非完整保护工具，而是筛选器的接口。

未诞大厅

所属层级：胎海层

地貌：像一座还没来得及决定用途的巨大殿堂，里面漂浮着未命名的结构、门、器官与人格残片。

资源：神性胚核、主导人格残片、初门组织、原始晦能。

威胁：人格抽离、未诞合唱、候选人格互噬。

剧情功能：终局抉择之地：成神，还是带着文明出走。

20灾变机制 · 黑潮与区域事件

黑潮不是天气，而是晦能世界的一次呼吸。它既是灾害，也是资源刷新、地图重组、规则松动与系统功能开放的时刻。

小黑潮 · 每七天一次

  • 怪物活动增强
  • 作物异变率上升
  • 门外模仿声增多
  • 频道里出现已死者发言
  • 潮后资源短时爆发

大黑潮 · 每三十天一次

  • 区域地形重组
  • 安全屋相对位置改变
  • 土著遗迹浮现
  • 玩家势力接壤
  • 系统开放新权限或新榜单

区域级事件

门缝季

所有门缝都“稍微宽了一点点”，敲门者类异常大幅活跃，门槛苔和门栓材价格暴涨。

集体同梦

整个区域梦见同一聚落，梦腐层入口因此暴露。那座聚落也可能梦见了他们。

活墙收屋

某聚落的墙开始吞并废弃安全屋，财富与异化同步增长。

旧灯回档

区域短暂进入上一轮外来者残响状态，玩家得以旁观前人如何失败。

未诞合唱

深层传来尚未成形人格的合唱，听懂一句，就会少一点“自己”。

21黑潮后稀有现象图鉴

黑潮带来的不只是怪物增强与地图重组。更珍贵、也更危险的是那些只在潮后短暂出现、仿佛世界自己露出内部结构的现象。

门影重叠

分类：潮后稀有现象 / 门类异常

现象：同一面墙上会短暂叠出两到三层门影。

出现条件：大黑潮后、门脉层波动期间、旧门遗迹附近。

价值：裂视者与旧门守墓者可借此寻找隐藏通路。

风险：进去的未必是离你最近的那扇门，而是离你“最像”的那扇。

旧灯回声雨

分类：潮后稀有现象 / 旧灯残响

现象：空气中会掉落细碎发亮颗粒，像一场不会打湿人的雨。

出现条件：大黑潮后、旧灯遗迹附近、梦腐层外溢时。

价值：可回收上一轮外来者残留的词句、情绪与短时记忆。

风险：接得太多，会开始怀疑自己是否也曾在上一轮死过。

空屋呼吸

分类：潮后稀有现象 / 活化建筑先兆

现象：某些废弃安全屋会在黑潮后恢复微弱呼吸、温度与居住感。

出现条件：骨沉层、废弃安全屋群、黑潮刚退后。

价值：是吞并、回收与活化研究的宝地。

风险：它恢复的不只是结构，还有“等主人回来”的执念。

潮后童声

分类：潮后稀有现象 / 遗留教育残响

现象：在没有幼体聚落的区域听见儿童说笑与认门歌。

出现条件：旧聚落遗址、废弃学校、土著儿童墓区附近。

价值：往往意味着附近有旧聚落、埋没村规或失落教育遗物。

风险：回应它们，可能会被当成“该回来上课的人”。

未诞光膜

分类：深层征兆 / 神胎结构裸露

现象：深层区域表面浮现像薄膜一样的微光。

出现条件：极稀有，多见于胎海层波动或大黑潮后期。

价值：可能是神胎深层结构暂时裸露的征兆。

风险：见过的人里，一部分疯了，一部分异常安静，还有一部分开始想把自己的房子养大。

潮后丰熟

分类：潮后农业现象

现象：某些作物在一夜间异常成熟，甚至长出平时不可能有的高品质变种。

出现条件：小黑潮后、污染与腐殖恰好平衡的农田。

价值：种田派与潮农派最渴望的事件。

风险：收成越漂亮，越要问一句——它到底吃了什么才长这么快。

地层翻梦

分类：梦腐层外溢现象

现象：地面像翻身一样轻微起伏，随后冒出大量不属于当前时代的梦渣片段。

出现条件：梦腐层浅表区、大黑潮后、旧灯梦库附近。

价值：可打捞旧灯时代生活碎片。

风险：有时梦被翻上来，人也会被“翻进别人的昨天”。

骨雨不落地

分类：骨沉层规则异常

现象：潮后有极细碎的骨屑从高处落下，却在离地不远处悬住。

出现条件：骨沉层大黑潮后、废骨气流密集处。

价值：可采集高纯度骨材与仪式材料。

风险：说明这片区域的“落下”概念短暂失效了，别在这里跳。

迟到的脚印

分类：路径残响现象

现象：黑潮结束后，地上会慢慢浮现并补出“昨天本该有却没出现”的脚印。

出现条件：黑潮沉积带、失踪事件后、旧路附近。

价值：可追踪失踪者、回门者和旧路残响。

风险：有些脚印会一路走到你门口，再突然停住。

井口开花

分类：水脉 / 边界外溢现象

现象：某些井口在潮后长出大量边界类或梦类植物。

出现条件：灰井修会旧址、井喉民聚落附近、黑潮后水脉活跃期。

价值：稀有资源喷发点。

风险：说明井下那一层正在主动把东西递上来。

屋内结露

分类：庇护所内部异常

现象：庇护所内部无故凝结黑露或灰露。

出现条件：活化度上升、黑潮刚退、生态槽失衡时。

价值：表示屋内已经开始参与更深层循环。

风险：也表示“屋子内部”不再完全属于你定义的内部。

归路发芽

分类：路径边界现象

现象：某条常走的回家路上突然长出成片门槛苔、守床草或名刺苔。

出现条件：长期重复通行路径、大黑潮后、庇护所稳定度高但活化度上升时。

价值：说明这条路的归属感极强，可做稳定路径。

风险：太强的归属感，有时也意味着这条路不希望你再走去别处。

22主角路线 · 与世界的关系

穿越者中的异类

他穿越前只是普通人，却在穿越瞬间看见了不可名状的“造主”。那一眼击碎了他的世界观，也拔高了他的审美。
他第一次真正想活，不是为了安全，而是为了接近、理解，甚至成为那样的存在。

两大精神锚点

记忆锚：见过真正震撼、美丽、不可名状之物，那记忆是他不被无光域完全同化的底线。

自我观察锚：他始终审视自身，确保自己不是被疯狂吞没，而是在主动雕刻疯狂。

普通玩家：搜资源 → 升级房屋 → 抵御黑潮 → 扩大生产。
主角：搜资源 → 观察规则 → 驯化异常 → 建立循环 → 养房成舟 → 反向利用世界。

工具

它能保护我，所以有用。

实验体

它在变异，我想看看边界在哪里。

合作者

它开始回应我的命令。

家

它想留住我，而我没有第一时间拆了它。

23世界真相分层

真相层级

第一层 · 求生
玩家以为这只是一场带系统的求生游戏。
第二层 · 生态
怪物、植物、庇护所与死者都参与同一晦能循环。
第三层 · 历史
上一轮外来者曾经来过，并留下失败的文明残响。
第四层 · 系统
系统不是救世工具，而是接入协议与筛选器。
第五层 · 神胎
整个无光域是一枚即将诞生的神胎。
第六层 · 候选人格
所有强者、榜首、裂视者和异常适应者，都是主导人格候选。
第七层 · 造主
神胎本身也未必是终点。对更高处而言，这一切可能只是一次不起眼的孕育实验。

三大核心矛盾

生存 ⇄ 被饲养

玩家越努力活下去，越可能成为更优质、更高回流的能量产出单位。

建设 ⇄ 同化

庇护所越强大，越接近晦能生态，也越不像原本的人类文明。

成神 ⇄ 出走

主角最终要决定：成为神胎的主导人格，还是养出一艘能带文明从胎体里逃走的方舟。

24第一卷 · 孤门之始

从孤门小屋到第一次大黑潮，在前三十天内建立基础生态循环，并确认庇护所、系统与无光域都不是求生舞台，而是孕育结构的一部分。

第一卷结尾钩子

【第一次区域融合完成。】
【领地功能开放。】
【请各位灯种，开始互相靠近。】

主角通过裂视看到第二行隐藏文字：
【请各位候选人格，开始互相吞食。】

无光域是一片没有恒星的宇宙空洞，由一种名为“晦能”的活性生命维持生态。
土著、怪物、植物、庇护所、梦境、门、系统与穿越者，都被纳入晦能反哺体系。
而这场全民穿越的本质，是神胎诞生前最后一次进食与人格筛选。

如果世界是子宫，那么主角想做的，不是成为胎中的神，
而是把房子养成一艘船，带着文明从黑暗的孕育里逃出去。
