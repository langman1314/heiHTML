06晦植图鉴

晦植不是普通意义上的植物。它们是无光域在地表与庇护所内部长出的器官，负责喂养、修补、记录、引诱、静默与固定边界。

粮食类

灰苔 骨麦 盲薯 夜脂果 腔豆 壳米穗 静浆果 盲果藤 骨盐藻

药性类

止语草 缝息苔 冷梦花 烬舌叶 返脉根 褪痛菌 灰肺苔 静针果 灰喉苔

建材类

灰木 骨藤 静皮树 壳苔 脊柱竹 裂膜草 井绳藤 倒根树 骨帘藤 胃灯果 血温藤

精神类

眠菇 门纹眠菇 孵梦眠菇 回声蕨 哀铃果 旧梦藻 食刻菇 井眠藻 哭铃草 壁听花 雾喉花

边界类

门槛苔 守床草 界钉花 名刺苔 誓根 门睫草 归箱草 埋名花 返床菇 褪名苔

污染 / 寄生类

噬影藤 手指芽 哭种 忘返树 伪灯花 腐脐花 血灰藓 潮后芽 驮壳菌
灰苔

分类：基础食用晦植

外观：灰绿色薄苔，常贴在潮湿墙角、石面与腐殖边缘。

生态：只要有轻微湿气与腐殖残留就能生长，是无光域最普遍的低级食物来源。

用途：可烘成苔饼，也可做菌床底料。

风险：长期食用会让人逐渐习惯腐味，并产生轻微幻听。

骨麦

分类：主粮作物

外观：穗细而硬，麦芒像一层密骨针。

生态：根系缠绕白骨而生，骨越完整、死因越浓烈，生长越快。

用途：稳定主粮，可磨粉、熬糊、做长期储粮。

风险：丰收季，整片麦田会低声讨论死者的事。

门纹眠菇

分类：精神晦植 / 规则研究菌

外观：菌盖上有天然门形裂纹，纹路清晰得像被刻出来。

生态：多生长在旧门遗迹、门缝异常区和高密度规则残留地。

用途：用于门类规则研究和旧门残痕追踪。

风险：食用后常梦见“门后站位”，并记住一扇并不存在的门。

孵梦眠菇

分类：精神晦植

外观：菌伞柔软下垂，伞缘像未睁开的眼皮。

生态：生于高梦渣环境和长期未完成的情绪残留区。

用途：能把未完成梦境继续做下去，产出高纯度梦渣。

风险：梦里的人物会逐渐记住你，并在下一次梦里先向你打招呼。

灰肺苔

分类：药性晦植 / 庇护所辅助植物

外观：灰白薄苔贴在墙面，潮湿时会微微起伏，像在呼吸。

生态：常出现在潮湿、孢子浓度高、通风差的房间和洞穴。

用途：过滤孢子与腐臭，为菌房净气，缓解轻度呼吸污染。

风险：贴满卧室后，住户会梦见自己在替房子呼吸。

灰喉苔

分类：声音处理植物

外观：像一层湿润的细褶薄肉，常贴在井壁与管道内侧。

生态：生于长期有声流经过、但需要被压低回音的空间中。

用途：覆盖井壁、管道或墙体，能把流经的声音磨钝，用于静音通道与缄默村内壁。

风险：待得太久，人会越来越懒得解释自己。

静针果

分类：药性 / 工具辅助植物

外观：细长灰果，剖开后果核坚硬如细针。

生态：多见于低风、低温的静区边缘。

用途：果核可做缝合钉，果肉有轻微麻醉效果。

风险：服用后短时间不容易分辨是伤势变轻，还是身体已经懒得提醒你了。

胃灯果

分类：灯具植物

外观：半透明偏黄，中空果壳表面布满细密静脉纹。

生态：常生长在稳定热源和弱晦光附近。

用途：晒干后可做天然灯罩，能柔化晦晶灯的刺激性。

风险：灯光过稳时，会吸引灯腹虫和某些追灯异常。

井眠藻

分类：精神 / 水脉植物

外观：深灰色细长藻丝，漂浮时像一缕缕湿发。

生态：只在梦污染较高但水脉尚未彻底腐坏的深井和暗渠中生长。

用途：稳定井脉梦污染，也被壳眠裔提炼为安睡药液。

风险：从井中捞起时，有时会听见藻丝滴落陌生梦话。

食刻菇

分类：精神晦植 / 遗迹附生菌

外观：菌盖灰白，表面会短暂浮现刻文。

生态：多见于遗迹墙边、骨板堆、旧门铭文附近。

用途：承载短暂刻文、提取遗迹残余信息、辅助裂视读取规则注释。

风险：有些文字不是留下来的，而是正在写给你。

门睫草

分类：边界晦植

外观：极细黑灰草丝，从门缝和窗缝中整齐长出，像一层睫毛。

生态：只生于被反复确认的开合边界。

用途：预警门外异常晦能流，监测敲门者类事件。

风险：若草尖全部朝内，说明庇护所内部已更危险。

誓根

分类：契约植物

外观：根系裸露打结，像一团细小手指彼此绞缠。

生态：只会在立誓、结盟、婚约、血契或长期交易地发芽。

用途：见证婚约、盟约、长期交易与多人聚落秩序。

风险：违约过多之地，誓根会长成像绞索一样的扭结。

归箱草

分类：仓储秩序植物

外观：短而密，草叶像一排排竖起的标签。

生态：多长在长期稳定、编号清晰的储物区角落。

用途：判断储物区分类是否混乱，提示箱体是否开始“忘记主人”。

风险：一旦成片枯死，通常意味着仓储已经孳生边界型晦诡。

埋名花

分类：失名植物

外观：花瓣层层向内卷起，像把中心藏住。

生态：生于失名墓地、无主床位、旧门残址与身份异常高发地。

用途：识别失名区与身份型异常痕迹。

风险：摘下后若不立刻重新命名周边事物，容易引发身份错位。

返床菇

分类：睡眠边界植物

外观：灰白小菇常沿床腿、床板下缘成圈生长。

生态：多出现在长期未处理、却仍保有归属感的旧床位下。

用途：它出现时，说明那张床仍被某种归属感牵着。

风险：通常意味着“这张床还在等谁回来”。

褪名苔

分类：边界 / 失名植物

外观：极浅灰白，像被磨薄的旧纸屑，贴在刻字表面蔓延。

生态：生长在旧门牌、墓碑、骨板与教团活动边缘。

用途：缓慢磨掉门牌、墓碑与骨板上的名字，是食名教团最喜欢的辅助植物之一。

风险：若它出现在你家门口，说明已经有人在尝试让这里不再属于你。

血温藤

分类：建材 / 寄生植物

外观：暗红偏灰，触感像温热湿绳。

生态：喜欢攀附热源、睡眠区与育养间边缘。

用途：可做保温绳、育养带与温床边框。

风险：离体后仍会主动往体温更高的地方爬。

驮壳菌

分类：共生运输植物

外观：扁平厚菌像一层层柔韧货架垫片。

生态：多贴附于背井龟、壳眠牛等大型驮运生物背部。

用途：帮助固定货物。

风险：生长过久后，它会把货物也算作壳体的一部分。

雾喉花

分类：战术植物

外观：花腔鼓胀，像含着一口永远吐不尽的雾。

生态：多见于低温裂谷和伏击频发的阴湿地带。

用途：捏破后释放短时遮蔽雾，可用于撤离和伏击。

风险：雾里藏得住你，也藏得住更早等在里面的东西。

骨帘藤

分类：边界 / 建材植物

外观：中空藤蔓节部坚硬，垂落时像成串骨帘。

生态：常生长在旧门和可居住边界附近。

用途：编成门帘可减弱敲门者、门缝笑声与轻微模仿音。

风险：长期浸在高活化环境里，会自己学会“开合”。

柒

07晦兽图鉴

晦兽不是随机刷新的怪物，而是晦能生态中的“动物位执行者”。它们知道该搬什么、吃什么、守什么、追什么，也知道什么时候该把人归类进资源表。

追声型

听骨犬、沼喉犬、缝面鹿、脊鸣鼬、灰褶鸮

食腐型

灰背搬蚁、腐灯蛾、伏脐兽、食槽獾、胃囊菌共生群

边界侵蚀型

门齿鼹、回门螳、锈门蟹、壁行螯、门腹蜈、缝背狸

运输 / 驯养型

壳眠牛、背井龟、井背蜥、灰乳兽、门乳兽、锈骨牛、眠脊猫

精神污染型

裂鸣鸦、拾声蛛、梦掘虫、扁墓鳐、负壳人蜗、梦壳螺、灯盲蝠

领主 / 精英型

骨冠猎母、雾育母鹿、吞架熊、门眠熊、烬壳蛛母、壳井母蛛
听骨犬

分类：追声型

外观：体型近犬，四肢紧实，头骨偏长，耳部退化。

习性：靠骨骼震动追踪生命活动，不依赖传统视觉。

用途 / 生态位：是白骨区和骨沉层最常见的“震动猎手”。

风险：厚软苔层、无骨泥地能扰乱它，但只要你踩回骨地，它就会重新“听见你”。

缝面鹿

分类：追声 / 模仿型

外观：头面平整无五官，表面像被粗糙缝起。

习性：会拼接熟人的语气、死者的呼唤和你最想听见的话。

用途 / 生态位：不急于猎杀，更擅长驱赶、误导和分离队伍。

风险：它不急着杀人，它更喜欢让你先主动靠近。

壳眠牛

分类：驯养型

外观：壳层覆盖背部与颈侧，眼部浑浊安静，体态沉稳。

习性：缓慢、稳定、耐惊扰，是商路与聚落最珍贵的运输生物之一。

用途 / 生态位：产黑乳，适合运输、育养、菌床补液与安神配方。

风险：长期依赖其乳制品的人，会越来越少做梦。

灰褶鸮

分类：静默预兆兽

外观：翼膜像旧布褶皱，常扁平地贴在梁上。

习性：平时几乎不动，常在有人“说错话”时睁眼。

用途 / 生态位：静听者和缄默村常观察它判断空间是否被错误命名。

风险：它不是发现声音，而是发现某句话终于把某种东西说清楚了。

门乳兽

分类：稀有驯养型 / 边界兽

外观：大型哺乳型晦兽，腹侧有天然护膜，乳腺发达。

习性：通常守巢，不轻易离开已认定的安全边界。

用途 / 生态位：其乳液可稳定幼崽情绪与小范围边界，是高等级庇护所与活化领地的重要培育素材。

风险：错误驯养会让房屋采用“巢穴逻辑”而非人类秩序。

嚼字猿

分类：遗迹破坏型

外观：中型群居兽，牙齿宽钝，舌头粗厚，手指灵活。

习性：专啃门牌、骨板、图纸、刻文与记录载体。

用途 / 生态位：它们粪便中常残留被消化不净的文字碎屑，可供规则研究。

风险：它消化的不是纸张，而是命名与信息结构。

背井龟

分类：运输型

外观：背壳中央自然凹陷，便于积水、固架和挂载。

习性：缓慢而稳定，能记住一条长期使用的回家路。

用途 / 生态位：长途运水、运载种植模块和迁移庇护所核心物资。

风险：它记得回家路，但不保证那条路黑潮后还是原来的路。

门眠熊

分类：精英型 / 旧门栖伏兽

外观：体型庞大，伏睡时像一扇发霉发黑的旧门。

习性：喜欢盘踞在废门堆、门灾遗迹和断链走廊附近。

用途 / 生态位：守着大量旧门残骸，是门脉类资源的重要高危守卫者。

风险：一旦惊醒，会沿着“你最想回去的那条路”追你。

梦壳螺

分类：精神污染型

外观：灰黑螺壳布满梦纹，贴耳可闻模糊呓语。

习性：缓慢爬行，喜欢停在睡眠区附近或梦污染富集处。

用途 / 生态位：壳中可保存睡眠残响，壳眠裔常用于安眠药材与梦境记录。

风险：夜里会回放最近吞下的最后一段梦话。

骨线雀

分类：小型收集兽

外观：细瘦灰雀，喙尖发白，脚趾像细骨钩。

习性：喜欢叼走发丝、骨丝、线绳、门牌碎屑与细小归属物筑巢。

用途 / 生态位：它的巢常成为寻找失物与追踪边界异常的线索点。

风险：追巢者可能发现自己丢的不只是东西。

井喉鹭

分类：水脉感知型

外观：脖颈极长，喉袋下垂，常立在井边一动不动。

习性：对井中回声和深层水脉波动极其敏感。

用途 / 生态位：灰井修会常用它判断井下是否安全。

风险：它起飞通常不是因为受惊，而是因为井下有什么东西想上来。

缝背狸

分类：边界侵蚀型 / 盗取兽

外观：背部皮毛像被粗糙缝起，尾巴细长灵活。

习性：专偷针线、名签、门绳、缝合物与床沿布条。

用途 / 生态位：常是边界秩序松动的前期征兆。

风险：它偷走的通常不是贵重物，而是“让某样东西仍属于你的那根线”。

箱腹蜥

分类：储物寄生型

外观：腹部鼓起中空，像一个天然小箱。

习性：吞食小型物资、钥匙、药瓶、种子包与名牌。

用途 / 生态位：它的巢穴常成为失物集中区，也可反向判断某处仓储是否混乱。

风险：它吐出来的东西通常还被“分门别类”得很整齐。

眠脊猫

分类：驯养 / 精神寄生边缘型

外观：细长安静，脊背柔软，常蜷在熟睡者背后。

习性：喜欢贴近稳定睡眠区，靠吸走噩梦与余温维生。

用途 / 生态位：能吸走部分噩梦，因此受许多新聚落欢迎。

风险：养久了，主人会越来越少做梦，也越来越少产生新的想法。

骨舟虫

分类：群体协作型

外观：白骨色小甲虫，单只不起眼，成群时像一层微微流动的壳。

习性：在腐泥、薄沼和湿烂地表群聚结阵。

用途 / 生态位：成群时能在腐泥与薄沼上结成“临时活桥”。

风险：桥过完就散，留在上面的人会一起沉下去。

灯盲蝠

分类：追灯型

外观：翼膜薄灰，眼部退化，只剩对稳定灯火的趋性。

习性：喜欢围扑无光灯与骨灯。

用途 / 生态位：是稳定灯区常见干扰者，亦可作为“照明暴露度”预警。

风险：靠近时会让人误以为灯自己在喘。

嗅誓犬

分类：契约感知兽

外观：鼻梁细长，胸骨突出，步态稳而克制。

习性：对誓根、名契与违约痕迹极其敏感。

用途 / 生态位：能闻出誓根、违约痕迹与伪装盟约。

风险：守约者爱它，骗子最怕它，而心里动过背叛念头的人同样不会喜欢它。

裂页鹫

分类：遗迹破坏型

外观：颈细翼阔，喙极适合撕裂纸页与皮卷。

习性：盘旋于旧书库、誊名库、公告残厅与遗迹文书堆上方。

用途 / 生态位：其羽骨与喙材偶尔可用于精密刻写工具。

风险：它们不会杀很多人，却能轻易杀掉一段历史。

捌

08晦诡图鉴

晦诡不是单纯怪物，而是尚未被稳定解释的规则现象。它们最可怕的地方，不是出现，而是它们常诞生于最自然的人类行为：补一句话、数一遍人数、加一副碗、确认一次床位。

传述型

敲门者、回声走廊、频道遗言、半句人、多余祷文、补全童谣、借火客

边界型

陌生室友、空箱占用者、漏记房间、多出来的门内侧、床尾坐者、回箱名签、门缝笑声

计量型

数不清的羊、第三串脚印、永远少一件、旧榜残名、无主碗、对数房间、提前脚步

身份型

饭桌缺席者、倒影迟到者、借名童、顺路者、晚归影、门牌旧姓、门后换班者

系统型

多出的任务、被删除的频道、错误资源标记、替睡者、错栏公告、排行残留、正午错觉、折返信使
敲门者

分类：传述型 / 门类晦诡

触发条件：夜晚谈论门外声音、庇护所边界松动、门区确认不足。

表现：在门外模仿熟人、家人或队友，请求进入。

处理方式：不回应具体身份，只重复确认边界：“门外不是家人，门内才是家。”

危险本质：它并不想破门，它想先取得“你承认门外也算自己人”的资格。

陌生室友

分类：边界型

触发条件：庇护所稳定度过低，多人共居秩序混乱。

表现：醒来发现房间里多了一张床，或多出一个昨晚“明明和你聊过天”的室友。

处理方式：立刻重新确认物品数量与床位归属，并使用守床草与规则尘标记。

危险本质：当“谁该住在这里”变得不清晰，世界就会主动帮你补一个答案。

替睡者

分类：系统型 / 身份侵入类

触发条件：连续守夜、极度疲惫、庇护所稳定度过低。

表现：人没睡，系统却判定你完成睡眠恢复；床铺有被睡过痕迹。

处理方式：重新确认床位归属，睡前命名，使用守床草与回声记录。

危险本质：有东西正在代替你完成“回家与睡觉”这件事。

半句人

分类：传述型

触发条件：在遗迹、频道或黑潮后反复提起残句。

表现：只说一句残话的前半句，逼迫听者补完后半句。

处理方式：不要替死人把话说完，不要在群体中尝试推测后文。

危险本质：一旦补完，整句话会成为新的小型规则。

晚归影

分类：身份型 / 跟随类

触发条件：长期独自返程、反复回头确认、精神持续紧绷。

表现：在返程最后一段路上多出一道影子，始终与你保持回家的距离。

处理方式：不与其并行对话，不回头确认，不让它先接近门区。

危险本质：当你习惯它存在的那天，它会先你一步回家。

空位同意者

分类：计量型 / 聚落规则异常

触发条件：多人讨论重大决策时留下一个无人认领的位置。

表现：那个空位会在某次表决中被默认为“也算在场”。

处理方式：重要议事时，不留空位；空位必须明确标注为“无人”。

危险本质：现实会承认那份表决，好像真的有人在那里点了头。

回箱名签

分类：边界型 / 仓储异常

触发条件：聚落频繁更换仓储编号、随意丢弃旧标签。

表现：已经丢弃的旧标签半夜重新回到箱子上，箱中物开始认旧主。

处理方式：旧签需焚毁并注销，归箱牌不得随意替换。

危险本质：仓储秩序会逐渐被过去的归属感夺回。

门后换班者

分类：身份型 / 边界型

触发条件：守夜交接不清，值守记录模糊。

表现：总有一个人声称“已经替你站过一班了”。

处理方式：交班必须留痕，守门人需明确口令与时段。

危险本质：如果你默认了这份好意，下一次它会默认你不需要再守门。

无主碗

分类：计量型 / 共食异常

触发条件：固定共食形成，但餐具管理含糊。

表现：总会多一只碗，并逐渐出现使用痕迹。

处理方式：宁缺碗，不补未知空位；餐具必须固定归属。

危险本质：最终它会补出一段集体都“记得”的伪历史。

睡前点名者

分类：边界型

触发条件：多人庇护所、情绪紧张、夜规紊乱。

表现：总有人提议“再确认一遍屋里有谁”。

处理方式：夜间不做重复清点，改用固定床签和归位规则。

危险本质：确认得越细，越容易多一个。

借火客

分类：传述型 / 边界型

触发条件：夜里、门外、稳定灯火与主人的善意。

表现：门外有人来借火，却从不要求进门。

处理方式：不可递送已定义为“家中温暖”的火种。

危险本质：它借走的不是火，而是你家“温暖只属于自己人”的定义。

折返信使

分类：系统型 / 传播型

触发条件：频道留言、纸质传信与遗迹公告互相干扰时。

表现：送出去的消息会自己回来，而且常常多一行。

处理方式：不要相信被“补充完整”的信息，特别是过于正确的那部分。

危险本质：那一行通常不是恶意，而是正确得让人想相信。

存档床单

分类：遗留型

触发条件：长期沿用旧床具、未清洗旧血迹或旧梦渣。

表现：某张床单会复现前任主人的睡姿、体温与伤势。

处理方式：旧床具必须净化、换签或封存。

危险本质：睡得越久，越像接着别人的日子活。

门缝笑声

分类：边界型 / 高危先兆

触发条件：门区稳定度下降、敲门者被过度谈论、门栓受损。

表现：不是敲门，而是先听见门缝里传来一声轻笑。

处理方式：立刻加固门区、停止回应、重新确认边界词。

危险本质：那笑声的含义是——它已经觉得自己被允许进来了。

错栏公告

分类：系统型

触发条件：系统权限波动、空白公告厅残痕、门脉层黑潮。

表现：公告文字串错栏，像两条信息挤在同一行里。

处理方式：不得单独阅读，需多人校验且保留原始记录。

危险本质：读懂串错栏的公告，可能会接到根本不属于你的任务或命运。

正午错觉

分类：系统型 / 集体认知异常

触发条件：门脉层系统波动后，某些区域会突然集体认为“现在像是正午”。

表现：所有人都会短暂放松夜规，仿佛此刻不需要谨慎。

处理方式：以晦相和门区状态校准，不凭“感觉中的时间”调整规矩。

危险本质：一旦大家都忘了夜规为何存在，那些只在“所有人足够谨慎时”才被压制的东西会醒来。

09晦民种族与土著文明

“晦民”只是穿越者对本土智慧生命的统称。事实上，无光域的智慧生命在身体、语言、聚落结构、婚俗、教育、职业和信仰上高度分化。

骨纺人

擅长编织骨丝、菌丝、纤维与边界织物，是聚落建筑者与商队修补者。

壳眠裔

以睡眠保存历史、以梦境保管记忆，掌握梦葬、借梦和梦渣产业。

缄面族

以沉默维持边界，厌恶多余言语，擅长处理传播型与传述型晦诡。

食名裔

感知名字重量与裂缝。一部分是契约见证者，一部分则堕入食名教团。

裂瞳古裔

能看见更多晦能结构，也更容易失稳。古老、稀少，是主角未来可能的镜像之一。

井喉民

掌握井脉、水路与井下通讯，用水与喉振构成另一套社会网络。

灰蜕人

经历活化却保留理智的边缘群体，知道最多关于“同化后仍能活”的事。

灯骨匠裔

骨灯制作与灯纹语专家，坚信被看见的边界比被说出的边界更稳。

潮拾人

黑潮后第一批进入沉积带的拾荒者，最熟悉短效高值资源的收集时机。
对穿越者而言，无光域是地狱。
对晦民而言，无光域只是祖辈留下的家。

信仰核心

边界崇拜

他们不崇拜光，而崇拜门槛、床位、归属、名字、井口和可重复确认的秩序。

沉默崇拜

沉默不是美德，而是防灾措施。说错一句话，就可能替世界多生一张嘴。

饥饿节律

不能一次喂饱世界。收成、祭品、尸体处理和集体庆典都必须分批进行。

归位观念

睡前数器物，醒后数器物。确认自己仍在自己的床、自己的屋、自己的名字里。

儿童与教育

半名期

新生儿不立即拥有完整真名，只用乳名或暂名。因为孩子不是纯洁，而是边界过薄，容易被世界过早记住。

第一次认门礼

孩子在长辈陪同下摸过家门、床脚、箱子和餐具，逐一确认归属。这是生存教育，不是礼仪。

守床草仪式

第一次独睡前，要在床边栽守床草，并念归位词，避免床位被别的存在先认走。

婚俗与亲缘

换静礼

缄面族婚俗。双方需在长时间沉默中共同完成一项边界劳动，如铺床、修门、挂帘。谁先开口，谁就不适合同居。

共梦礼

壳眠裔婚俗。两人需在见证人记录下做一场稳定的共享梦，若梦境互相排斥，婚约作废。

编界礼

骨纺人婚俗。共同编织一道门帘或床沿边界，意味着愿意一起维持一个家。

借名礼

食名裔正统派婚俗。只交换一小部分名音，表示允许你进入我的边界，但不允许你吞掉我的位置。

11土著器物与日常用品图鉴

真正让世界活起来的，不只是怪物与灾变，还有人们在这种世界里如何吃饭、睡觉、记账、守门、结婚、葬人和记住彼此。

骨灯

最常见的土著照明器。灯罩由骨、胃灯果或馁灯芯构成，既照路，也传递灯纹语。

门绳

挂在门内侧的绳结装置，用于确认门是否被动过，也提醒守门人换班。

床签

挂在床脚的名签或纹骨片，用于确认床位归属。多人聚落里比武器更重要。

骨板账册

誊名师与箱契师使用的记录工具，不只记数量，还记“谁的、放哪儿、为什么在这里”。

静皮帘

由静皮树皮制成，用于隔音、隔话和隔梦话，是睡眠区与审议室常用品。

归位碗

共食时固定分配给某人的碗具。长期混用容易诱发身份与计量类异常。

守夜骨铃

夜间巡逻提示与交班器具，铃声极轻，却能在静皮帘后稳定传播。

静听盘

静听者用来贴墙、贴门、贴井壁，判断另一边是否有异常回声的工具。

失名布

覆盖死者面部、旧门牌、危险遗物和疑似身份污染物，暂时隔绝“被继续认领”的风险。

井纹壶

井喉民储水、运水与判水器具，壶身纹路会对水脉状态产生微弱回应。

封床钉

用于对异常死者、失踪者或高危回归者的床位进行封锁。

换静帘

缄面族婚俗与深夜议事时使用，象征“帘内之言不出帘外”。
